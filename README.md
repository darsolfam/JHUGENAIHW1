# JHUGENAIHW1

This repository contains the work for JHU Generative AI Assignment 1, completed by Daren J Spencer. It demonstrates using a coding assistant (Claude Code) to draft a personal introduction file, followed by manual editing to make the content authentic.

## Git steps used

1. Initialized a local git repository with `git init`
2. Created the three required files and made an initial commit
3. Used Claude Code to generate content for `assistant_draft.md` and committed
4. Manually edited `assistant_draft.md` into `final_version.md` and committed
5. Connected the local repo to GitHub with `git remote add origin` and pushed with `git push -u origin main`
